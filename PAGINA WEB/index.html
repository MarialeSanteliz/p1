Como Empezar Una Buena Rutina De Skincare

Limpiador Facial

Es importante utilizar un buen limpiador facial dependiendo tu tipo de piel. Para las pieles grasas un gel limpiador es ideal y para pieles secas una leche de limpieza

Crema Hidratante

La crema hidratante no puede faltar en tu rutina de skincare. Las pieles grasas tambien neecsitan hidratacion, para este tipo de piel una crema en gel y liviana es un plus. En cambio las pieles secas necesitan de cremas mas pesadas para evitar la resequedad.

Protector Solar (FPS)

Por ultimo y no menos importante el protector solar es indispensable en una rutina, evita las manchas y las arrugas. Debes usarlo todos los dias y retocar cada 3 horas.

Productos De Nuestra Linea Actual de Makeup

En nuestra linea de maquillaje vas a poder encontrar bases, correctores, blushes, contornos, mascaras de pestañas, etc. Contamos con diferentes colores para cada color de piel.
